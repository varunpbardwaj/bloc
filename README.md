## Installation

- Open the Extensions sidebar in VS Code
- Search for **Bloc**
- Click Install
- Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
- Select **Preferences: Color Theme** and pick a Bloc variant

## Themes

| Theme | Notes |
|-------|-------|
| **Bloc Ash** | Neutral gray |
| **Bloc Stone** | Warm gray |
| **Bloc Slate** | Cool gray |
| **Bloc Carbon** | High contrast |
| **Bloc Midnight** | Blue-charcoal |
| **Bloc Sand** | Warm with beige |
| **Bloc Graphite** | Neutral dark |
